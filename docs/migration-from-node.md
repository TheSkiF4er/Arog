# Миграция с Node.js на Farog

Farog предоставляет инструмент `farog import-node`, упрощающий перенос существующих проектов.

## Общий процесс

1. Скопируйте проект на основе Node.js.
2. Установите Farog.
3. Выполните `farog import-node`.
4. Разрешите найденные несовместимости.
5. Запустите тесты и приложение под Farog.
